# Personal Website

This context describes the visitor-facing areas and concepts of Levin's personal website.

## Language

**Portfolio**: The area that presents Levin's background and work, including about, experience, education, projects, achievements, bookmarks, and social profiles.

**GitHub repository**: The public GitHub repository associated with a portfolio project. _Avoid_: Source repository, source URL

**Live site**: The publicly accessible running version of a portfolio project. _Avoid_: Demo, deployment

**Blog**: The area where visitors discover and read Levin's published articles.
